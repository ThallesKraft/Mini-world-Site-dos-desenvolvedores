Como editar scripts?

Basta seguir estes passos:

Etapa 1: criar um mapa no modo de desenvolvimento

Etapa 2: clique no último ícone no canto

Etapa 3: clique no botão de script esquerdo para abrir o editor de script

Etapa 4: clique no conjunto de scripts à direita e clique em Novo script para editar seu script.

Passo 5: Não se esqueça de salvar clicando no ícone salvar no canto superior esquerdo. Mude para o modo de reprodução para executar o script

Limpar script

Como excluir todos os scripts de uma só vez?>


Na interface de edição de script, clique no ícone da lixeira para limpar todo o script atual.
